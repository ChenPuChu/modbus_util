# CRC

`标准CRC生成多项式如下表`

| 名称     | 生成多项式                   | 十六进制    |
|--------|-------------------------|---------|
| CRC-4  | $x^4+x+1$               | 0x13    |
| CRC-8  | $x^8+x^5+x^4+1$         | 0x131   |
| CRC-8  | $x^8+x^2+x+1$           | 0x107   |
| CRC-8  | $x^8+x^6+x^4+x^3+x^2+x$ | 0x15E   |
| CRC-12 | $x^{12}+x^{11}+x^3+x+1$ | 0x180B  |
| CRC-16 | $x^{16}+x^{15}+x^2+1$   | 0x18005 |

## CRC-16

* 左移16位
* 取前16与校验码异或
* 将异或结果与后n位拼接后，再次与校验码异或
* 直至位数小于16
